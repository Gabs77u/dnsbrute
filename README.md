# dnsbrute
The dnsbrute application is a subdomain brute force tool. It is used to find hidden subdomains of a web application.

The application works as follows:

The user provides a web application URL and a list of words.
The application checks whether the URL is valid.
The application checks whether the word list is empty.
The application iterates over the list of words and checks whether each word is a subdomain of the provided URL.
The dnsbrute application can be used for security or exploitation purposes.
